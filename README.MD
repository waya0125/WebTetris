# Web TETRIS

## Description

This is a simple Tetris game. It is based on the [Tetris game](https://github.com/ophum/tetris)

## Installation

1. ZIP Download or Git Clone this repository.
2. Open `index.html` in your browser.
3. Enjoy.

## Usage

| Key     | Action       |
|:--------|:-------------|
| A/←     | Move Left    |
| D/→     | Move Right   |
| W/Space | hard drop    |
| S/↓     | soft drop    |
| Q/Z     | Rotate Left  |
| E/↑     | Rotate Right |
